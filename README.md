# Emu8086-ile-g-kta-ndan-ka-ma-oyunu
Asteroid Escape - 8086 Assembly Game  Intel 8086 mimarisi üzerinde, x86 Assembly dili kullanılarak geliştirilmiş gerçek zamanlı bir kaçış oyunudur. Proje; düşük seviyeli donanım programlama, DOS/BIOS kesmeleri (INT 10h, 16h, 1Ah) ve doğrudan video belleği erişimi gibi temel sistem programlama prensiplerini uygulamalı olarak sergilemektedir.
